# About Airdrop Hunter

## What is Airdrop Hunter?

Airdrop Hunter is an automated intelligence tool for Web3 crypto enthusiasts who want to stay on top of airdrop opportunities without spending hours on research.

## Two Versions Available

| Version | Platform | Repository | Status |
|---------|----------|------------|--------|
| **MCP Server** | Antalpha MCP Server | [antalpha-com/antalpha-skills](https://github.com/antalpha-com/antalpha-skills) | Active |
| **Coze Skill** | Coze/OpenClaw | [AntalphaAI/airdrop-hunter](https://github.com/AntalphaAI/airdrop-hunter) | Legacy |

The **MCP Server version** is the recommended version. It provides 5 structured tools (airdrop-scan, airdrop-daily-report, airdrop-check-project, airdrop-zero-cost, airdrop-scam-check) and fetches data directly from DefiLlama API with no search plugins required.

## The Problem

- **Information Overload**: Hundreds of airdrop opportunities flood Twitter, Discord, and Telegram daily
- **Time-Consuming Research**: Manually checking 10+ sources takes 2+ hours daily
- **Scam Risk**: Phishing sites and rug pulls are common
- **Missed Opportunities**: High-value airdrops often have short windows

## The Solution

Airdrop Hunter automates the entire research process:

1. **Scans** protocol data from DefiLlama API
2. **Filters** out completed airdrops (exact match prohibited tokens)
3. **Grades** projects by risk/reward (S/A/B system) using VC funding data
4. **Delivers** a concise daily action checklist

## Who Should Use This?

- **Airdrop Hunters** who want to maximize efficiency
- **Crypto Newcomers** who don't know where to start
- **Busy Professionals** who can't spend hours on research
- **Risk-Averse Users** who need security warnings

## Key Differentiators

| Feature | Airdrop Hunter | Manual Research |
|---------|----------------|-----------------|
| Time Required | 5 min/day | 2+ hours/day |
| Risk Assessment | Automated grading | Manual evaluation |
| Security Alerts | Built-in warnings | Must verify yourself |
| Missed Deadlines | Automated reminders | Easy to miss |
| Data Source | DefiLlama API (structured) | Scattered web sources |

## How It Works

```
DefiLlama API -> Protocol data (TVL, category, chains)
       |
Funding Database -> VC backing, funding amounts
       |
  Grading Engine -> S/A/B/C classification
       |
  Daily Report -> Structured action checklist
```
